Prediksi Curah Hujan Menggunakan Metode Pembelajaran Mesin Berdasarkan Data Historis Selama 20 Tahun¶

Curah hujan merupakan salah satu komponen penting dalam sistem iklim yang berdampak signifikan pada kehidupan manusia, ekosistem, dan berbagai sektor ekonomi. Informasi akurat tentang pola curah hujan sangat dibutuhkan, terutama untuk perencanaan sektor pertanian, pengelolaan sumber daya air, mitigasi bencana banjir, dan keberlanjutan lingkungan. Ketidakpastian dalam distribusi curah hujan, terutama akibat perubahan iklim global, menimbulkan tantangan besar bagi pengambilan keputusan yang efektif di berbagai sektor tersebut.

Kemajuan teknologi dalam pengumpulan data dan pengembangan model prediktif berbasis kecerdasan buatan (AI) dan pembelajaran mesin (ML) menawarkan solusi potensial untuk memperkirakan curah hujan dengan tingkat akurasi yang lebih tinggi. Pemanfaatan data historis seperti suhu, kelembapan, tekanan atmosfer, arah dan kecepatan angin, serta data spasial dan temporal lainnya memungkinkan pengembangan model prediksi yang lebih kompleks dan dapat diandalkan.

Tujuan:¶

Tujuan utama dari pengembangan model prediksi curah hujan adalah:

  1. Meningkatkan akurasi prediksi untuk mendukung pengambilan keputusan.
  2. Memitigasi dampak bencana hidrometeorologi seperti banjir dan kekeringan.
  3. Mendukung keberlanjutan sektor pertanian dan pengelolaan sumber daya air.
  4. Memberikan wawasan yang lebih mendalam terhadap dinamika iklim regional dan global.

Rumusan Masalah:¶

  1. Bagaimana membangun model prediksi curah hujan yang akurat dengan memanfaatkan data meteorologi historis dan teknologi kecerdasan buatan?
  2. Apa saja faktor meteorologi yang paling signifikan memengaruhi curah hujan, dan bagaimana hubungan antara faktor-faktor tersebut dapat dimodelkan?

# Fungsi untuk membuat sliding windows
def create_sliding_windows(data, window_size):
    X, y = [], []
    for i in range(len(data) - window_size):
        X.append(data[i:i + window_size])
        y.append(data[i + window_size])
    return np.array(X), np.array(y)

# Menyiapkan data untuk sliding windows
window_size = 3  # Ukuran window
data_values = df[['Temperature', 'Precipitation']].values  # Menggunakan kolom 'Close' dan 'High' untuk prediksi
X, y = create_sliding_windows(data_values, window_size)

# Membuat DataFrame untuk hasil sliding windows
sliding_window_df = pd.DataFrame(X.reshape(X.shape[0], -1), columns=[f'Temperature_t-{window_size-i}' for i in range(window_size)] + [f'Precipitation_t-{window_size-i}' for i in range(window_size)])
sliding_window_df['Target_Temperature_t'] = y[:, 0]  # Target Close
sliding_window_df['Target_Precipitation_t'] = y[:, 1]  # Target High

# Menampilkan hasil sliding windows
print(sliding_window_df.head())

    scaler_features.fit_transform(sliding_window_df.iloc[:, :-2]),  # Ambil semua kolom kecuali target
    columns=sliding_window_df.columns[:-2],  # Nama kolom tanpa target
    index=sliding_window_df.index
)

# Normalisasi target (Target_Temperature_t dan Target_High_t)

    scaler_target.fit_transform(sliding_window_df[['Target_Temperature_t', 'Target_Precipitation_t']]),
    columns=['Target_Temperature_t', 'Target_Precipitation_t'],

)

# Membuat plot untuk perbandingan hanya Random Forest
plt.figure(figsize=(10, 6))
plt.scatter(y_test, rf_y_pred, alpha=0.5, label='Random Forest', color='blue')
plt.plot([y.min(), y.max()], [y.min(), y.max()], 'r--', label='Identity Line')  # Garis identitas
plt.xlabel('Nilai Aktual')
plt.ylabel('Nilai Prediksi')
plt.title(f'Plot Nilai Aktual vs Prediksi (Random Forest)\nMSE: {rf_mse:.2f}')
plt.legend()
plt.grid()
plt.show()
